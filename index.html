About the Project

System Utility Software is a Windows utility application designed to help monitor and manage your system. It provides several built-in tools that allow users to view system information, manage files, and interact with system commands in one place.

Features

  • Dashboard: Displays CPU usage and allows report generation.
  • Files: Built-in file manager to create, edit, delete, and manage files.
  • Console: Run PowerShell commands directly within the application.
  • System Info: Displays operating system details and CPU information.
  • Text Editor: Edit .txt files directly inside the program (Windows only).
