System Utility Software

A Python-based utility tool developed by Pam687 also known as Parmartha Joshi

About the Project

This software helps users optimize and manage their system efficiently. It’s built with Python and packaged as an executable for easy use.

Features

  • Dashboard: Shows CPU usage and allows you to generate reports.
  • Files: Built-in file manager to create, edit, delete, and manage file locations.
  • Console: Run PowerShell commands directly inside the software.
  • System Info: Displays OS information and CPU usage.
  • Text Editor: Edit .txt files directly inside the program (Windows only).
