System Utility Software

A Python-based utility tool developed by Pam687 also know as Parmartha Joshi

About the Project

This software helps users optimize and manage their system efficiently. It’s built with Python and packaged as an executable for easy use.

Features

  • Dashboard: The dashboard shows your CPU usage and has a button to create a report for it.
  • Files: The files tab is a in-built file manager, you can create edit and delete files, and you can also change the location of saving your file.
  • Console: The Console tab allows you to run powershell commands without even opening Windows Powershell, but expect some bugs because this option is new.
  • System Info: The system info tab shows your OS and CPU usage and more, expect some bugs here too because the OS might not be based on real-time statistics.
  • Text Editor (only for windows computers): The in-built text editor (This only for Windows computers) allows you to choose .txt files on your computer and edit change or delete them, in sooner updates I'll make this multi-OS.
